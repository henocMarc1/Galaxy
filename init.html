Instructions pour l'admin

Pour créer un compte admin, créez d'abord un compte utilisateur via la page d'inscription.

Ensuite, allez dans la console Firebase → Realtime Database → users → votre UID → changez "role" de "customer" à "admin".

Vous pourrez ensuite accéder au panneau admin : admin.html
